Eatery & Coding Community

A full-stack website concept for an online eatery, allowing users to create and edit accounts, book tutoring sessions, and order menu items. The front-end features and layout of each website page were designed using CSS3, Bootstrap, HTML5, and Javascript. This website is programmed using full-stack CRUD functionality to manage queries through the use of Python3 logic, models, and modularization.
